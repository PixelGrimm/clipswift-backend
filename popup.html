❓ ClipSwift Help & Tips

🚀 Getting Started

Create Snippets: Click the + button to create a new snippet. Give it a command (like :hello) and content.

Auto-completion: Type your command (e.g., :hello) in any text field on the web, and it will automatically expand to your content!

⚡ Quick Commands

  • + - Create new snippet
  • 🤖 - AI Content Generator
  • ☀️/🌙 - Toggle light/dark theme
  • ? - This help menu

🎯 Auto-completion Tips

  • Commands must start with : (e.g., :welcome, :thanks)
  • Auto-completion works instantly as you type
  • Edit commands and content - changes apply immediately
  • Works on any website with text fields

🤖 AI Content Generator

  • Describe what you want to say
  • Provide a command (e.g., :welcome)
  • AI generates professional content for you
  • Perfect for customer service, greetings, and responses

📁 Organization

  • All: View all your snippets
  • Favorites: Quick access to starred snippets
  • Categories: Organize by AI Prompts, Templates, Messages, Code
  • Search: Find snippets quickly

💡 Pro Tips

  • Use descriptive commands like :customer_support or :bug_report
  • Create templates for common responses
  • Star your most-used snippets for quick access
  • Use categories to keep your snippets organized
